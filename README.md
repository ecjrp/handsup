# handsup
Hands up / hands down via command for RP servers

# Installation
> * Upload **handsup** to your server resources folder
> * Add - **handsup** to your AutoStartResources in citmp-server.yml Restart and enjoy.

# Usage
> * /hu - puts hands up.
> * /hd - puts hands down.
